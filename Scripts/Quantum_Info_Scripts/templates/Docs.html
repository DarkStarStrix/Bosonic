Documentation

The Quantum Algorithm Suite provides a simulation engine for running quantum algorithms. Here's how to use it:

  1. Go to the Home page.
  2. Select a script from the dropdown menu. The scripts represent different quantum algorithms.
  3. Click the "Run Simulation" button. The system will run the selected script and display the results on the canvas.

The system handles all the complexities of running the simulation and visualizing the results. All you need to do is select a script and click "Run Simulation".
